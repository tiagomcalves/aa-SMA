# Foraging:
## Environment:
2D grid with walls/obstacles, a starting point and a destination point.

## Agents: 
Must move and avoid obstacles. Can be a multi-agent system (e.g. 2 agents look for the way out, one helps another).

## Objective:
Find the shortest path or the way out in the minimum steps possible  
(cooperation or competition if there is limited resources on the path).
